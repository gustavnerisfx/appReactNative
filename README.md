# RESENHA FLIX

<p>Este é um projeto feito para um trabalho de PAM na etec Fernando Prestes.</p>
<p>O app consiste em um simples "letterboxd" de resenhas únicas para filmes, séries e animes.</p>

## Linguagem usada

- JavaScript
- React Native Framework
- Expo
